    My Story

    My name is Percy Njuguna. I am a rising programmer who started with a love for computers, followed by curiosity about the coding world. I began by writing small bits of code in Notepad and trying to run them.

    Instead of running, the code flashed red error messages on the screen, making me lose hope and believe programming was not for me.

    Later, I downloaded VS Code, which made learning easier. My code finally worked, and I started building an arcade-style website called Cold Fire. Never lose hope—even when results are not clear.
