# My Professional Portifolio

Welcome to my personal portfolio website! This project showcases my skills in web development, my journey as a technical developer, and the projects I have built while learning HTML, CSS, and JavaScript.

## 🚀 Features
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Clean Navigation:** Easy access to Home, About, and Contact pages.
* **Project Showcases:** Direct links to my latest web development work.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend/Hosting:** Firebase & GitHub Pages
* **Tools:** VS Code, System CMD (Git)

## 📁 Project Structure
* `index.html` - The main landing page.
* `about.html` - Information about my background and skills.
* `contact.html` - Ways to reach out to me.
* `styles.css` - Custom styling for the entire site.
* `services.html` - The services that I offer.
* `blog.html` - A technical journal documenting my projects, coding challenges, and my progress in learning Spanish.
* `projects.html` - A curated gallery of my web development work and GitHub repositories.
